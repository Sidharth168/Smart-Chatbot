# Smart-Chatbot


# 🤖 Smart Chatbot Assistant

A web-based chatbot built using **Python**, **Flask**, **HTML**, **CSS**, and **JavaScript**. The chatbot provides interactive conversations through a simple and responsive web interface using a rule-based response system.

---

## 📌 Project Overview

The Smart Chatbot Assistant is designed to simulate basic human conversation. Users can type messages into the chatbot, and it responds based on predefined intents and patterns stored in a JSON file. This project demonstrates the fundamentals of chatbot development, web application development, and frontend-backend communication.

---

## 🚀 Features

* Interactive chatbot interface
* Rule-based conversation system
* Flask backend
* Responsive web design
* JSON-based intent and response management
* Easy to extend with AI models like Google Gemini

## ▶️ Installation

1. Clone the repository

```bash
git clone https://github.com/Sidharth168/Smart-Chatbot.git
```

2. Navigate to the project folder

```bash
cd Smart-Chatbot
```

3. Install the required packages

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

5. Open your browser and visit:

```text
http://127.0.0.1:5000
```
